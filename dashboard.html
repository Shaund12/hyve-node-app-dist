💡 How Upgrades Work

When a governance proposal for a chain upgrade passes, the node halts at the specified block height and requires a new binary.

1. Click Download Latest Binary to fetch the upgrade binary from the Hyve release server.

2. Click Apply Upgrade & Restart to swap the binary and restart the node.

3. If there are issues, use Rollback to restore the previous binary.
